Reference Guide
VARIABLES:
  • x : Joystick steering (-1.0 = left, 1.0 = right)
  • y : Joystick throttle (-1.0 = reverse, 1.0 = forward)
  • toeangle : Active maximum toe-in angle (radians)
  • transweight : Translation mixing weight
  • rotweight : Yaw rotation mixing weight
  • latweight : Sideways lateral mixing weight
  • pi : Math constant π
